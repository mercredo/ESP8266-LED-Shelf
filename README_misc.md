# ESP8266 LED Shelf

This version is a fork of [Winston-Lu's ESP8266-LED-Shelf](https://github.com/Winston-Lu/ESP8266-LED-Shelf), an improved version of [this LED shelf by DIY Machines on Youtube](https://www.youtube.com/watch?v=8E0SeycTzHw) to include more features and settings. [Link to the original Github repo here](https://github.com/DIY-Machines/DigitalClockSmartShelving)

This fork aims at providing LED clock countdown features soon.
